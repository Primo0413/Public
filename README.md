# WPFSerialAssitan 串口助手

## 引言

**一款基于C#及WPF的串口助手软件。本串口助手操作方便，UI简洁。软件实现了基本的串口通讯需要的功能，开发者可以基于此添加自定义的功能。此外，可以基于此开发出一些简单的串口控制类软件。**

**08.20 移植开源代码（上述串口助手）对此代码进行学习以及自定义修改，后续代码更新简介放在此README中，并且删除不必要的关于源代码的描述，源代码可从下方项目主页中寻找**

## 项目主页

* [基于WPF开放源码的串口调试助手(https://github.com/ChrisLeeGit/SerialAssistant)](https://github.com/ChrisLeeGit/SerialAssistant)

## 基本功能

* 串口数据接收，可以采用多种模式显示接收到的数据：字符，十六进制等
* 串口数据发送，可以手动/间隔自动发送
* 保存串口接收到显示区的数据
* 保存/加载软件配置
* 独创的“简洁视图模式”， 便于用户专注于数据的接收和发送
* 08.21新增自定义帧校验格式以及帧校验插入位置（如选择插入位置则对插入位置之前的字节进行校验）

## 使用帮助

## 使用到的开源库

* 本软件使用的一个开源的Json.Net库，可以非常方便地使用它用Json格式存储配置信息或者加载Json格式的配置文件。其主页为：http://www.newtonsoft.com/json。
